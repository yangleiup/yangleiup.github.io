# 水平居中

# 1. 若是行内元素，给其父元素设置 text-align:center,即可实现行内元素水平居中

<style>
  body{
    background-color: #e9e9e9;
  }
  .wrap{
    background-color: yellow;
    width: 200px;
    height: 200px;

    text-align: center;
  }
</style>

# 4. CSS3 transform,该元素设置如下:

<style>
  body{
    background-color: #e9e9e9;
  }
  .parent{
    background-color: yellow;
    width: 200px;
    height: 200px;
  }
  .child{
    position:absolute;
    left:50%;
    transform:translate(-50%,0);
  }
</style>
