# 用生动有趣的emoij美化你的commit log

先来看看下面仓库的 commit log（来自 Antd仓库），使用了很多 emoij表情。

#

使用方法 每个 emoji 都有自己对应的代码，你只需要在编写 commit log 时带上这个代码就可以了，比如：

  • 🎉：初次提交 对应代码: :tada:

  • 🎨：改进代码结构/代码格式 对应代码： :art:

  • 🐛：修复bug 对应代码：:bug:
